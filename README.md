# 📍 個人旅遊地圖小筆記 (Travel Map Notes)

---

## 🎯 專案定位
這是一個專為「個人」設計的簡易旅遊筆記系統，強調 **Local-first (本地優先)** 與 **完全免費**。所有資料儲存在瀏覽器中，並可透過 Firebase 免費方案進行部署。

---

## 🚀 核心功能需求

### 1. 左側管理面板 (Sidebar)
*   **國家管理**：可以新增、列出國家清單。
*   **地點清單**：選擇國家後，下方顯示該國家內所有的旅遊地點。
*   **連動功能**：點擊地點清單中的項目，地圖會自動移動至對應的標註點 (Marker)。

### 2. 地圖操作 (Map Interaction)
*   **標註點連動**：地圖上標註的點位會自動同步顯示在左側「地點清單」中。
*   **小筆記 CRUD**：
    *   **新增**：在地圖上點擊右鍵或按鈕新增標註點與小筆記。
    *   **修改**：點擊 Marker 開啟資訊視窗，直接修改筆記內容。
    *   **刪除**：可移除不再需要的標註點。
    *   **查詢**：透過關鍵字搜尋地圖上的筆記內容。

### 3. 技術規範 (完全免費方案)
*   **前端框架**：Vue 3 (Vite)。
*   **地圖引擎**：Leaflet.js + OpenStreetMap (不需 API Key，完全免費)。
*   **本地儲存**：IndexedDB (持久化儲存大容量資料)。
*   **雲端備份/部署**：Firebase Hosting (提供免費 HTTPS 網址)。

---

## 📂 專案結構建議
*   /src/components/Sidebar：處理國家與地點清單。
*   /src/components/Map：地圖核心物件與 Marker 管理。
*   /src/store：使用 Pinia 或簡單的 Reactive Object 管理本地狀態。
*   /src/services/db.ts：IndexedDB 封裝邏輯。
